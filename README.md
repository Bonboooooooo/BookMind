# BookMind 图书查询平台

BookMind 是一个图书查询与数据可视化平台，当前已完成前端页面搭建与 GitHub Pages 在线部署。

## 当前已完成功能

- 完成 BookMind 前端页面设计
- 支持输入关键词进行图书查询
- 使用前端模拟数据展示查询结果
- 已通过 GitHub Pages 部署为可访问网页

## 后续计划

- 接入真实图书数据
- 使用 RDS MySQL 存储图书信息
- 使用函数计算 FC 开发后端查询接口
- 实现图书关键词查询、分类筛选、分页展示
- 使用 ECharts 实现图书数据可视化
